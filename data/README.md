# Data directory

The raw and processed data in this project are generated by the R scripts.

Run the scripts in this order from the repository root:

```r
source("R/00_setup.R")
source("R/01_define_grammar.R")
source("R/02_generate_corpus.R")
source("R/03_compute_surprisal.R")
```

Generated files include:

- `raw/artificial_corpus.csv`
- `processed/token_surprisal.csv`
- `processed/critical_surprisal.csv`
- `processed/critical_by_corpus_block.csv`

The first three files are not tracked in the default repository because they are large and can be reproduced with the fixed random seed in `R/00_setup.R`.
